# **App Name**: SyncStream

## Core Features:

- Landing Page: Full-screen background with sign-in options (Google, Anonymous) and footer links.
- Signup/Avatar: Pixel avatar generator/selector with a display name input form.
- Chat/Channels Page: Main chat screen with channel list, invite functionality, and settings.
- Channel Creation and Invitation: Allows users to create new channels and invite other users using a shareable link.
- Real-time Chat: Real-time messaging within channels, including text and YouTube link recognition.
- Music Sync: Synchronized music playback in channels using YouTube URLs; manages isPlaying and updatedAt via Firebase.
- Music Suggestion Tool: Tool which assists users to choose appropriate music to play next, in response to user requests. Takes into account the listening history of channel members to help curate music.

## Style Guidelines:

- Primary color: Teal (#45b3a9) for a modern and clean look, inspired by listen-together.app.
- Background color: Dark grey (#282c34) for a sleek and immersive experience.
- Accent color: Cyan (#64c5eb) for interactive elements and highlights to guide user attention.
- Headline font: 'Poppins' (sans-serif) for titles and prominent text.
- Body font: 'PT Sans' (sans-serif) for channel descriptions and other supporting text; pairs well with Poppins.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets or URLs.
- Minimal, line-based icons for navigation and actions.
- Mobile-first, clean UI with rounded corners on cards and buttons.
- Subtle CSS animations for hover and tap effects to enhance interactivity.
